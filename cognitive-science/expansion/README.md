# 50 个拓展认知范围的方向

这个专题把意识、自由意志、制度、财富、复杂系统、技术未来等 50 个方向写成独立文章。每篇文章都尽量避免模板化重复，围绕主题自身展开：

- 核心概念是什么？
- 它改变了什么理解？
- 它能解释哪些现实问题？
- 它在什么地方容易被误用？
- 它如何进入日常判断和行动？

## 写作约定

- 长篇（> 4000 字）：适用于主题已经有充分学术研究、概念需要被科学依据锚定的方向（如 03-cognitive-bias、04-emotion-function、46-information-overload、48-technology-cycle）
- 中篇（2000–4000 字）：适用于已有清晰框架但需要展开的方向
- 短篇（< 2000 字）：仅适用于概念本身已经足够清晰、需要保留扩展空间的方向

任何一篇若被读者认为深度不足，应优先补足：定义精确性、实证数据、机制解释、可执行的应对策略、可引用文献。
