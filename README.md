## 认知科学

跨学科深度分析，从数学本质到工程实践的认知体系：复利原理（数学→物理→控制论→复杂网络→AI系统）、知识网络拓扑与自催化回路、压力管理的控制论框架与SRE工程映射。

## Spring Cloud

从 L1 到 L30 的完整微服务课程体系。当前进度：10/30。
涵盖：Nacos 服务注册发现、OpenFeign、LoadBalancer、Sentinel 熔断降级、Spring Cloud Gateway、配置中心、链路追踪、分布式事务、K8s 部署等。

## 知识复利的理念

这个仓库本身是知识复利理念的实践：
- 每一条笔记成为新知识的锚点
- 跨学科连接产生交叉验证的自催化效应
- 间隔回顾确保净增长率为正

> **知识的量不重要，知识的内化结构和连接密度才重要。**
