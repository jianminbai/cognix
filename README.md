# cogniX

cogniX 是一个认知科学与个人知识复利实验室，用来沉淀注意力、学习、元认知、决策质量、问题选择、压力管理和长期成长等方向的深度分析。

它不是单纯的资料仓库，而是一套持续迭代的认知系统：每篇文章都尽量保留问题背景、核心模型、实践框架和后续连接，让知识可以被回看、复用、迁移和再次生长。

## 知识复利原则

- 每一条笔记都要成为新知识的锚点，而不只是临时摘录。
- 优先建立跨学科连接，用一个领域的模型验证另一个领域的问题。
- 保留可执行的实践框架，让知识可以落到学习、工作和长期选择里。
- 通过间隔回顾、索引重组和文章互链，提高长期内化率。

> 知识的量不重要，知识的内化结构和连接密度才重要。
