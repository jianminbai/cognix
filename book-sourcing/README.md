## 找书流程

1. **搜索源优先级**：lunarora.com > Sogou 搜索 > 多多软件站 > ebook-treasure-chest
2. **格式优先级**：epub > azw3(zip) > pdf > mobi
3. **下载方式**：城通网盘（境外可直接下载，限速约16KB/s）、夸克网盘（境外不能直下，需国内登录）
4. **清理策略**：确认投送飞书后立即删除临时文件

## 经验总结

- 城通网盘 <3MB 文件下载无问题，>10MB 可能损坏
- 夸克网盘只能发链接给用户自行下载
- Sogou 搜索返回真实搜索结果（百度验证码、Bing 空白、Google 超时）
